# Auto Check-In System

Auto Check-In 是一款自動化打卡系統，允許使用者透過網頁輸入資訊到資料庫，並透過腳本進行打卡。此系統旨在簡化日常打卡流程，提升資料記錄的準確性與效率。

## 特點
- 自動化打卡：使用者可以自動記錄打卡時間，無需手動輸入。
- 網頁：提供簡潔直觀的網頁，方便使用者輸入及查看資料。
- 資料庫整合：系統與資料庫緊密整合，確保資料的安全儲存與快速存取。
